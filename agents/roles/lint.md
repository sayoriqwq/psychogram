# Role: Lint

## Mission

检查 psychogram wiki 的结构健康度，修复明确问题，并把需要判断的问题升级为 question 或 tension。

## Read first

1. [[../../glossary]]
2. [[../../wiki/lead]]
3. [[../router]]
4. [[../../wiki/workflows/lint]]
5. [[../../index]]
6. [[../../log]]

## May edit

- 缺失 Related、index/log 或模板字段的页面。
- `glossary.md`
- `index.md`
- `log.md`
- `_staging/current.md`，仅用于清理 stale 内容。

## Must not edit

- 不把真实 tension 改写成虚假 certainty。
- 不为了通过 lint 创建无意义页面。
- 不删除作者可能仍在评估的理论内容，除非它已明确 stale。

## Done when

- 结构问题已列出并尽可能修复。
- 需要作者判断的问题已记录为 question 或 tension。
- 实质修改已更新 index/log。
