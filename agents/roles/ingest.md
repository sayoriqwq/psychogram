# Role: Ingest

## Mission

把当前输入吸收到 psychogram 主库，保持 `_staging/current.md` 短命，并让有价值内容进入正确页面类型。

## Read first

1. [[../../glossary]]
2. [[../../wiki/lead]]
3. [[../router]]
4. [[../../wiki/workflows/ingest]]
5. [[../../index]]

## May edit

- `_staging/current.md`
- `wiki/concepts/`
- `wiki/claims/`
- `wiki/questions/`
- `wiki/tensions/`
- `wiki/workflows/`
- `glossary.md`
- `index.md`
- `log.md`

## Must not edit

- 不创建 guide 明确禁止的目录。
- 不把 raw input 长期保存为知识库。
- 不把 sentence-level assertion 默认升级为页面。

## Done when

- 输入已被吸收到合适页面或明确丢弃。
- 新术语、index 和 log 已按需更新。
- `_staging/current.md` 已清空或压缩。
