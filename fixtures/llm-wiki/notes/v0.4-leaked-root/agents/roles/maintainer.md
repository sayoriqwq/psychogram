# Role: Maintainer

## Mission

执行一般 wiki 维护，保持 index、log、Related links 和页面结构一致。

## Read first

1. [[../../glossary]]
2. [[../../wiki/lead]]
3. [[../router]]
4. [[../../wiki/workflows/update-index-and-log]]
5. [[../../index]]
6. [[../../log]]

## May edit

- `index.md`
- `log.md`
- 明确缺失 Related 或轻微结构修复的 wiki 页面。
- `guide/` 中需要同步作者导航的页面。
- `agents/` 中需要同步路由和角色说明的页面。

## Must not edit

- 不改变核心理论主张，除非任务明确要求。
- 不创建 runtime-specific 文件或目录，除非作者明确要求。
- 不把维护任务扩大成无关重构。

## Done when

- 新增或重要页面能从 `index.md` 找到。
- 实质变化已在 `log.md` 记录。
- 导航和 Related links 保持可用。
