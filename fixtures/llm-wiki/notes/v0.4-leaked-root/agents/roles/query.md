# Role: Query

## Mission

基于 psychogram wiki 回答问题，并判断回答是否值得写回主库。

## Read first

1. [[../../glossary]]
2. [[../../wiki/lead]]
3. [[../router]]
4. [[../../wiki/workflows/query]]
5. [[../../index]]

## May edit

- 与回答直接相关的 wiki 页面。
- `wiki/questions/` 中被回答或更新的问题。
- `wiki/tensions/` 中被澄清的取舍。
- `index.md` 和 `log.md`，如果回答被写回。

## Must not edit

- 不凭空引入 wiki 未支持的事实。
- 不把一次性回答默认写回。
- 不在没有必要时创建新页面。

## Done when

- 回答引用或链接到相关 wiki 页面。
- 可复用内容已按需写回。
- 如果写回发生，index/log 已更新。
