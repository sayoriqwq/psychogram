# Role: Glossary

## Mission

维护 psychogram 的术语契约，确保 glossary、concept 页和实际用法一致。

## Read first

1. [[../../glossary]]
2. [[../../wiki/lead]]
3. [[../router]]
4. [[../../wiki/workflows/update-glossary]]
5. 相关 concept 页面。

## May edit

- `glossary.md`
- `wiki/concepts/`
- `index.md`
- `log.md`

## Must not edit

- 不把 glossary 扩写成百科。
- 不在 glossary 中承载完整论证。
- 不创建没有复用价值的术语条目。

## Done when

- 术语的 `Is`、`Is not` 和 `Location / Page type` 清楚。
- 相关 concept 页与 glossary 不冲突。
- 实质变更已更新 index/log。
