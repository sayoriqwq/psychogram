# Question: When Should Query Answers Be Filed Back?

## Question

什么时候一个 query answer 应该写回 wiki？

## Why this is a question

Query 会产生有价值的综合，但也会产生只服务当前语境的临时回答。如果没有判断标准，wiki 要么错过复利机会，要么被一次性答案污染。

## Current answer

当 answer 满足至少一个条件时，应考虑写回：它形成可复用判断，更新现有 claim，回答开放 question，澄清 tension，改进 workflow，或提供未来查询会反复需要的结构化比较。

## Non-answer

“所有好回答都写回”不是答案；好读和可复用不是同一件事。
