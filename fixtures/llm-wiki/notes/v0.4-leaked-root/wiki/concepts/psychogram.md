# Psychogram

## Definition

Psychogram 是一个由 agent 维护、由作者在 Obsidian 中阅读和判断的理论 wiki 系统。它把当前输入和持续讨论吸收到稳定、互联、可维护的 Markdown 主库。

## Is not

它不是普通笔记库、聊天记录仓库、长期 raw source 层，也不是每次 query 时临时综合的 RAG 结果。

## Role in psychogram

这是系统整体名称，也定义了作者和 agent 的分工：作者判断方向和意义，agent 负责维护、链接、写回和检查。

## Common confusions

- 把 psychogram 当成 notes folder，会导致目录无限扩张。
- 把 psychogram 当成 runtime agent 项目，会让 projection 反向支配理论结构。
- 把所有输入都永久保存，会削弱主库的稳定性。
