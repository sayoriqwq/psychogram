# Workflow: Ingest

## Purpose

Maintain this LLM-wiki concept wiki by absorbing source or discussion input.

## Steps

1. Read `glossary.md`, `wiki/lead.md`, `index.md`, and target pages.
2. Identify new or changed concepts, claims, questions, tensions, or content workflows.
3. Update affected wiki pages.
4. Update glossary if terms change.
5. Update index and append log.

## Done when

- New input is reflected in wiki content.
- Index/log are current.
