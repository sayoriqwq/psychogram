# Question: When Should Good Answers Be Filed Back?

## Question

When should a query answer become part of the wiki?

## Why this is a question

Some answers create reusable synthesis; others only serve the current moment. Filing back too little loses value, while filing back too much creates clutter.

## Current answer

File back an answer when it clarifies a durable claim, resolves or sharpens a question, exposes a tension, or creates reusable structure.

## Non-answer

"Every good answer should be filed back" is too broad.
