# LLM-wiki

## Definition

LLM-wiki is a pattern for using an LLM to incrementally build and maintain a Markdown wiki.

## Is not

It is not just uploading files and asking questions over them.

## Role in this wiki

This page anchors the fixture output and explains the concept being modeled.
