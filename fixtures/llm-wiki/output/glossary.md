# Glossary

## LLM-wiki
**Is:** A pattern where an LLM incrementally builds and maintains a persistent Markdown wiki.  
**Is not:** A one-shot file upload answer or a generic notes folder.  
**Location / Page type:** [[wiki/concepts/llm-wiki]]

## persistent wiki
**Is:** A maintained knowledge layer that survives across sessions and compounds through updates.  
**Is not:** Temporary chat memory.  
**Location / Page type:** [[wiki/concepts/persistent-wiki]]

## schema
**Is:** The instructions that make an LLM act as a disciplined wiki maintainer.  
**Is not:** Merely a database schema.  
**Location / Page type:** [[wiki/concepts/schema]]

## raw source
**Is:** Original material used as input for wiki maintenance.  
**Is not:** The maintained wiki itself.  
**Location / Page type:** [[wiki/concepts/raw-source]]

## index file
**Is:** Content-oriented catalog for navigating the wiki.  
**Is not:** A chronological log.  
**Location / Page type:** [[wiki/concepts/index-file]]

## log file
**Is:** Chronological semantic record of wiki evolution.  
**Is not:** A content catalog.  
**Location / Page type:** [[wiki/concepts/log-file]]
