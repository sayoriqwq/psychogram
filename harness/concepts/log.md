# Log

## Definition

Log 是语义演化时间线，记录架构决策、fixture 发现、template 更新和 projection 变化。

## Is not

它不是 git diff，也不是全量操作流水账。

## Role in psychogram

Harness log 记录系统如何演化；generated concept wiki 的 log 记录该 wiki 的演化。

## Common confusions

- Fixture observation 应先记录在 fixture notes，再把泛化结果写入 harness log。
- Log 记录语义事件，不记录每个小编辑。
