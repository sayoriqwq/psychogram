# Staging

## Definition

Staging 是 generated wiki 内部的短期输入吸收区。

## Is not

它不是 harness 的知识层，也不是长期 capture 层。

## Role in psychogram

Template 提供 `_staging/current.md`，让每个 generated wiki 可以暂存当前输入并尽快吸收到内容区。

## Common confusions

- Harness 本身不需要根级 `_staging/`。
- Staging 中的材料只有进入 generated wiki 后才成为稳定内容。
