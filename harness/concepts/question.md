# Question

## Definition

Question 是会影响结构、维护或投射的开放问题。

## Is not

Question 不是普通待办，也不是两个合理需求之间的持续取舍。

## Role in psychogram

Harness question 用来记录系统层尚未稳定的判断点。

## Common confusions

- 每个 question 必须说明为什么它是真问题。
- 具体 concept 的问题属于 fixture output 或 instance。
