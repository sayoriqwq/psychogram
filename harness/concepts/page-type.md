# Page Type

## Definition

Page type 是 concept wiki 中承载稳定语义的页面类别，例如 concept、claim、question、tension。

## Is not

它不是任意目录扩张，也不是为了分类而分类。

## Role in psychogram

Page type 帮助 agent 判断新内容应该进入哪里，也帮助作者在 Obsidian 中理解 wiki 的结构。

## Common confusions

- 只有跨多个 concept wiki 都稳定需要的页面类型才应进入 template。
- 具体领域的特殊页面类型应由 instance 自己决定。
