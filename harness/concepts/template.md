# Template

## Definition

Template 是生成 concept wiki 的 domain-neutral 空骨架。它提供稳定目录、维护协议、基础 guide 和 agent workflow。

## Is not

Template 不是示例 wiki，不包含具体领域概念、主张、问题或张力。

## Role in psychogram

Template 让每个新 concept wiki 从同一套最小结构开始，同时避免把 fixture 内容复制进新 instance。

## Common confusions

- Template 可以有占位说明，但不能有领域样例页。
- Template 的 `agents/workflows/` 是维护流程，不是 `wiki/` 内容。
