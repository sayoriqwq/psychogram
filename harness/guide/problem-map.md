## Working answers

- 只有跨多个 concept wiki 都稳定需要的结构、规则、workflow 和 agent role 才属于 harness。
- Template 应保留最小可运行骨架，不带领域内容。
- Fixture 可以暴露问题，但只能把可泛化的观察提升为 harness 规则。
- Runtime 功能应通过 projection 表达，不进入 harness 核心。
