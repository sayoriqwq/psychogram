# Workflow: Update Glossary

## Purpose

Maintain glossary as a term contract.

## Steps

1. Add a term only when it recurs, affects maintenance, or is easy to confuse.
2. Write `Is`, `Is not`, and `Location / Page type`.
3. Keep entries short.
4. Use concept pages for expanded explanation.
5. Update index/log after substantial glossary changes.

## Done when

- Terms are clear and bounded.
- Glossary does not become an encyclopedia.
