# Claim: Good Query Answers Should Be Filed Back

## Claim

有复用价值的 query answer 不应只留在聊天中，而应写回 wiki，成为后续理论建设可继承的材料。

## Why it matters

Query 往往会产生新的比较、判断、连接和临时答案。如果这些内容不写回，它们会从系统记忆中消失；如果写回得当，wiki 会因为提问本身而增长。

## Challenges

- 并非所有回答都有长期价值。
- 回答写回太积极会污染主库。
- 某些回答只是对当前上下文的局部解释，不应升级为稳定页面。
