# Question

## Definition

Question 是仍缺答案、但会影响理解、设计或行动的问题。每个 question 页必须解释为什么它是真问题。

## Is not

Question 不是普通待办，也不是两个合理需求之间的持续取舍；后者应进入 tension。

## Role in psychogram

`wiki/questions/` 保存开放问题和当前答案。它帮助作者和 agent 看见理论哪里仍然不稳定。

## Common confusions

- 写不出 `Why this is a question` 的问题，不应建页。
- 一个 question 可以与多个 tension 相连，但它的方向是寻找当前答案。
