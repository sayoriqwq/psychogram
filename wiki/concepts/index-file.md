# Index File

## Definition

`index.md` 是内容导向 catalog。它列出重要页面、链接和一行说明，帮助作者和 agent 快速定位材料。

## Is not

它不是时间线，不是变更日志，也不是自动生成数据库。

## Role in psychogram

Agent 在 query 和 lint 前应先读 `index.md`，再进入相关页面。新增或大改页面后，必须更新 index。

## Common confusions

- Index 应服务内容定位，不应复制 guide 的阅读路径。
- Index 可以人工维护，不需要一开始引入 Dataview 或脚本。
