# Staging

## Definition

Staging 是当前尚未吸收到 wiki 的短期材料区。它承接聊天上下文、NOTE.md、临时想法和本轮 agent 工作中的 pending items。

## Is not

它不是长期 capture 层，不是 distill 层，也不应该演变成另一个知识库。

## Role in psychogram

`_staging/current.md` 让 agent 有一个临时工作台，但目标始终是清空、压缩或把有价值内容迁移到 `wiki/`。

## Common confusions

- “暂时放一下”很容易变成长期堆积。
- staging 中的内容只有被吸收到主库后，才成为 psychogram 的稳定知识。
