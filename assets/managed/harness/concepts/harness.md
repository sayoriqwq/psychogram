# Harness

## Definition

Harness 是 psychogram 本体。它定义通用规则、artifact class、page type、workflow、agent role、fixture lint 和 projection 边界。

## Is not

Harness 不是具体 concept wiki，也不保存 fixture 的领域内容。

## Role in psychogram

Harness 负责回答“如何生成和维护 concept wiki”，而不是回答某个概念内部的问题。

## Common confusions

- Fixture 观察可以提升为 harness 规则；fixture 内容不能提升为 harness 内容。
- Harness 中的 claim 应治理系统，而不是治理某个具体概念。
