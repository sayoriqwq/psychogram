# Agent Role

## Definition

Agent role 定义 agent 在某类维护任务中的使命、读写范围、禁止事项和完成标准。

## Is not

Agent role 不必等同于 runtime subagent 或某个平台配置。

## Role in psychogram

Role 让维护行为可路由、可审计，并避免 agent 把系统层、template 层和 fixture 层混在一起。

## Common confusions

- 语义 role 可以投射成 runtime 能力，但 projection 不是 role 的源头。
- Role 应服务 workflow，而不是替代 workflow。
