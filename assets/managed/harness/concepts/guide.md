# Guide

## Definition

Guide 是面向作者的导航层，帮助重新进入系统或某个 generated wiki。

## Is not

Guide 不是 agent runtime 配置，也不是全量 catalog。

## Role in psychogram

Harness guide 面向 psychogram 作者；generated wiki guide 面向该 concept wiki 的作者。

## Common confusions

- Guide 组织阅读路径。
- Index 组织内容目录。
