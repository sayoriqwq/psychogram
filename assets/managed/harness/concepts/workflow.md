# Workflow

## Definition

Workflow 是 agent 可执行的维护流程。

## Is not

Workflow 不等于概念内容本身，也不必立即投射成 runtime skill。

## Role in psychogram

Harness workflow 管理系统本体；generated wiki 的 agent workflow 管理该 wiki 的维护；概念内容区只在主题本身需要时讨论 workflow。

## Common confusions

- `harness/workflows/` 是系统流程。
- `template/agents/workflows/` 是 generated wiki 的维护流程。
- `template/wiki/` 不应默认包含 workflow 内容页。
