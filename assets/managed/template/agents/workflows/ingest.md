# Workflow: Ingest

## Purpose

Absorb current input into the concept wiki.

## Steps

1. Read current input from chat, note files, or `_staging/current.md`.
2. Identify candidate concepts, claims, questions, and tensions.
3. Update existing pages before creating new pages.
4. Update glossary if new shared terms appear.
5. Update index and append log after substantial changes.
6. Clear or reduce staging.

## Done when

- Valuable input is filed into the right wiki pages.
- Staging no longer holds processed material.
- Index and log reflect substantial changes.
