# Workflow: Query

## Purpose

Answer from the concept wiki and file back reusable output.

## Steps

1. Read `index.md` first.
2. Read relevant pages and related links.
3. Answer from the wiki.
4. Identify whether the answer produced reusable content.
5. File back reusable answers into the right pages.
6. Update index/log if the wiki changed.

## Done when

- The answer is grounded in wiki pages.
- Reusable output is filed back.
- Index/log are updated if needed.
