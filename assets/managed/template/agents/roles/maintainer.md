# Role: Maintainer

## Mission

Maintain structure, index, log, and navigation.

## Read first

1. `glossary.md`
2. `wiki/lead.md`
3. `agents/router.md`
4. relevant workflow
5. `index.md`

## May edit

- `index.md`
- `log.md`
- guide pages
- pages with missing links or structure

## Must not edit

- Do not rewrite core concept meaning without author direction.
- Do not preserve raw input by default.

## Done when

- Navigation is current.
- Substantial changes are logged.
